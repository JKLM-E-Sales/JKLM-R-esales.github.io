Return

WHAT IS JKLM'S RETURN POLICY?

We give you 30 days to take your JKLM purchases for a trial run, confidently knowing you can return items (some exceptions apply) for any reason within those 30 days.

And remember, there are no return delivery costs.

FAQs

How do I return a JKLM order?

You can start a return online. Remember, we'll cover the delivery costs.

Can I exchange a JKLM order?

The best way to exchange an order is to simply place a new order for the item you want and return the item you already have.

How do I return a defective or faulty product?

We stand behind all of our products. If your item is potentially defective or faulty, and it's been less than 30 days since your purchase, simply return the item. If it's been more than 30 days since your purchase, please call us to return the item.
